How to setup Game in devc++ - 

1. Download devc++ from the above link.
2. Install SFML. (Check the above link to setup)
3. Add the maingame.cpp file inside your devc++ folder (in C drive).
4. Add all the .wav files provided in the GameFiles folder to your devc++ folder.
5. Now you are good to go to enjoy the game.s
